# HangerGames - Application Mobile

## Description

HangerGames est une application mobile développée avec **React Native** et **TypeScript** qui permet de :  

- Afficher une liste de jobs ou de films (selon les données JSON).  
- Consulter les détails d’un job/film avec informations complètes.  
- Ajouter ou retirer des jobs/films aux **favoris**.  
- Rechercher des jobs par poste ou entreprise.  
- Gestion des données via **Redux Toolkit** pour les favoris.

L’application utilise **react-native-paper** pour une interface moderne et agréable.

---

## Fonctionnalités

1. **Liste des jobs**  
   - Affichage de tous les jobs disponibles.  
   - Photo de l’entreprise en image circulaire.  
   - Aperçu du poste, de l’entreprise et de la localisation.  

2. **Détails d’un job**  
   - Fiche complète avec description, adresse, téléphone, salaire annuel et date.  
   - Titres en gras et image centrée pour un rendu clair.  

3. **Favoris**  
   - Ajouter ou retirer un job/film aux favoris.  
   - Affichage du nombre de favoris sur le bouton dédié.  

4. **Recherche**  
   - Barre de recherche pour filtrer les jobs par ville.  

---

## Technologies utilisées

- **React Native**  
- **TypeScript**  
- **React Navigation** pour la navigation entre écrans  
- **Redux Toolkit** pour la gestion d’état  
- **react-native-paper** pour les composants UI  
- **JSON** pour les données statiques (jobs et films)
